Behavior Summary

  • Theme — Light / Dark / System segmented buttons. Selection saved to DataStore, instantly applies via Compose recomposition (no restart)
  • Language — Arabic / English segmented buttons. Uses AppCompatDelegate.setApplicationLocales() — Android restarts the activity gracefully
  • Notifications toggle — reflects current OS permission state. Tapping opens Android app notification settings (Intent to system settings)
  • On app start — POST_NOTIFICATIONS permission dialog shown once. If denied, silently skipped
  • Settings entry — gear icon (⚙️) in HomeScreen top app bar
